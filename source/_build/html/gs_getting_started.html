Getting started with Clear Linux* OS for Intel® Architecture¶

The easiest way to give Clear Linux OS for Intel Architecture a spin is to download a recent KVM image from the image directory on our download page. This directory contains a current image, the UEFI firmware helper and the KVM start helper script. See Running in a virtualized environment for details.

You can also Installing the OS as host on a target computer.
