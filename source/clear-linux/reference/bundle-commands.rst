.. _bundle-commands:

Useful bundle commands
######################

To see a list of currently-installed bundles, enter:

.. code-block:: console

   # swupd bundle-list

To see the list of all available bundles, enter:

.. code-block:: console

   # swupd bundle-list --all

To add a bundle, enter:

.. code-block:: console

   # swupd bundle-add [bundle name]
