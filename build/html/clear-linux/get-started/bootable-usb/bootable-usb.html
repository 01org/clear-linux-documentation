Create a bootable Clear Linux USB drive

Instructions to create a Clear Linux* OS for Intel® Architecture USB drive vary depending on your operating system.

Download the latest Clear Linux image

There are 2 types of Clear Linux images suitable for burning onto and running off a USB drive:

  • Live image: clear-[version number]-live.img.xz
  • Installer image: clear-[version number]-installer.img.xz

Go to the Clear Linux image repository and download the desired type.

With the appropriate image downloaded, choose the step-by-step instructions applicable to your system:

  • Create a bootable USB drive on Linux
  • Create a bootable USB drive on Windows
  • Create a bootable USB drive on macOS
